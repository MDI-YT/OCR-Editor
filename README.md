# OCR-Editor
 An amazing tool for OCR and editing and Converting handwritten text to docx or pdf(Format) as per the user wish.

Requirements:-
1) Pillow (PIL)
2) Pytesseract
3) Opencv-python 
4) Numpy 
5) Docx2pdf 
6) Python-docx
